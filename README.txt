== Description ==
    Multiple Currency Switcher for WooCommerce plugin that allows customers to change product prices and currencies according to current exchange rates in real-time and pay in their preferred currency.
    Multiple Currency Switcher for WooCommerce plugin that allows adding any currency to the WooCommerce store.
    The ideal solution to make the serious WooCommerce store site in multiple currencies!

== IMPORTANT NOTICE ==

The plugin works based on the WooCommerce plugin.

= Features of Multiple Currency Switcher for WooCommerce =

The plugin is bundled with the below features.

== Settings ==

= enabling/disabling WooCommerce Currency Switcher =

* Check Out Page Currency Switcher.
* Single Product Page Currency Switcher.
* Fixed Price Rate Currency Switcher.
* Auto Fixed Price Rate Currency Switcher.
* Default Currency.

= Add a new country for the switcher to change Currency =

* Select the Country option
* Select the currency position option
* Add Custom currency number of decimals
* Auto Detect rate: currency rate is auto Detect to select the currency
* Set the exchange rate manually: Multiple Currency Switcher for WooCommerce also allows you to set up the exchange rate manually.
* Manage the Exchange rate: Every time you exchange money from one currency to another,  The difference between the exchange rate they decide to apply and the real exchange rate traded in the market is the currency exchange fee.
* Unlimited Currency Add: you can add as many currencies as you want. With the free version.
* Update exchange rate automatically: The plugin automatically updates exchange rates. if you can select default currency to automatically update all currency exchange rates.

== Installation ==

1. Install Multiple Currency Switcher for WooCommerce by uploading the `multiple-currency-switcher-for-woocommerce` directory to the `/wp-content/plugins/` directory.
2. Activate the plugin through the `Plugins` menu in WordPress
3. Setup multiple currency switcher for woocommerce option by going to the `Settings > Multiple Currency Switcher` menu.
